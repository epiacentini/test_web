Top Quiz

Top Quiz is a Java quiz application with the front-end built out with JSwing. The quiz is comprised of 20 questions relating to American history and come in three varieties, short answers, multiple choice and interactive. At the end the user score is recorded and added to a hi-scores. The questions are the same but they are always in a different order.
